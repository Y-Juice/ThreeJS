Rentaspeed

RentaSPEED Is a usecase website for threejs as a 3d showroom to view all the different available cars


Features

3D Car Models: Explore 3D models of each car available for rent.

Color Customization: Change the color of cars in real time to match your preferences.

Modern Design: A sleek, user-friendly design that combines functionality with aesthetics.

WebGL Compatibility: Ensures broad browser support for rendering 3D graphics seamlessly.


Getting Started

Prerequisites

A basic understanding of JavaScript, HTML, and CSS.

A modern browser with WebGL support (check compatibility here).


Installation

1. Clone the repository:

```git clone https://github.com/Y-Juice/ThreeJS.git```
```cd ThreeJS```


2. Install dependencies:

```npm install```


3. Start the development server:

```npm run dev```
